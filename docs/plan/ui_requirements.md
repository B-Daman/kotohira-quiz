## 1. 現状の課題

現行のWeb UIは以下の固定仕様で、ユーザーが遊び方を選べない。

| 項目 | 現状 | 課題 |
|------|------|------|
| 出題構成 | 琴平5問 + 英語5問 = 固定10問 | 琴平だけ解きたいユーザーに英語が邪魔 |
| 難易度 | 全難易度からランダム | 初心者にhardが出て挫折 / 上級者にeasyばかりで物足りない |
| カテゴリ | 全カテゴリからランダム（各1問まで） | 特定テーマを深掘りしたいニーズに応えられない |
| 連続プレイ | 10問終了→スコア表示で終了 | 「もう10問やりたい」が叶わない |
| スタート画面 | なし（即出題開始） | モード選択の余地がない |

## 2. 画面構成

### 2.1 スタート画面（新規）

10問を始める前に、出題モードを選ぶ画面。

```
┌────────────────────────┐
│   ⛩️ 琴平クイズ         │
│                        │
│  ┌──────────────────┐  │
│  │  🎯 フリーモード   │  │  ← デフォルト。琴平10問ランダム
│  └──────────────────┘  │
│  ┌──────────────────┐  │
│  │  📚 難易度えらび   │  │  ← easy / medium / hard を選択
│  └──────────────────┘  │
│  ┌──────────────────┐  │
│  │  🏷️ テーマえらび   │  │  ← カテゴリを選択
│  └──────────────────┘  │
│  ┌──────────────────┐  │
│  │  🔤 英語もまぜる   │  │  ← 従来の琴平5+英語5モード
│  └──────────────────┘  │
│                        │
│  📊 これまでの成績 →    │
└────────────────────────┘
```

**設計方針:**
- デフォルトは「フリーモード」（琴平10問、全難易度・全カテゴリからランダム）
- 英語はオプション扱いに降格（現行はメイン扱いだが、琴平クイズが主役）
- ボタンを押すとすぐ出題開始（サブ設定が必要なモードのみ1ステップ追加）

### 2.2 難易度えらび（サブ画面）

```
┌────────────────────────┐
│  📚 難易度えらび        │
│                        │
│  ┌──────────────────┐  │
│  │  😊 やさしい       │  │  ← easy問題のみ
│  └──────────────────┘  │
│  ┌──────────────────┐  │
│  │  🤔 ふつう         │  │  ← medium問題のみ
│  └──────────────────┘  │
│  ┌──────────────────┐  │
│  │  😤 むずかしい     │  │  ← hard問題のみ
│  └──────────────────┘  │
│  ┌──────────────────┐  │
│  │  🎲 おまかせ       │  │  ← easy3:medium4:hard3 くらいの構成
│  └──────────────────┘  │
│                        │
│  ← もどる              │
└────────────────────────┘
```

- 「おまかせ」は easy:medium:hard をランダム比率で混ぜる（毎回違う構成で飽きにくい）
- 問題プールが足りない場合は他の難易度で補完する

### 2.3 テーマえらび（サブ画面）

2階層構造。トップレベルをタップで即出題、`>` のあるものはタップでサブカテゴリに展開。

```
┌──────────────────────────────┐
│  🏷️ テーマえらび              │
│                              │
│  ⛩️ 金刀比羅宮    274問  >   │  ← タップで展開
│  📜 歴史・人物    176問  >   │  ← タップで展開
│  🎭 金丸座・歌舞伎  55問     │
│  🗺️ 地理・観光    138問  >   │
│  🍜 グルメ・名産品  64問     │
│  🎨 文化・伝統     68問     │
│  🎪 祭り・イベント  82問     │
│  🏘️ 現代の琴平     91問  >   │
│                              │
│  ← もどる                    │
└──────────────────────────────┘
```

展開例（金刀比羅宮をタップ）:
```
│  ⛩️ 金刀比羅宮    274問      │
│    ├ すべて        274問      │  ← 全体から10問
│    ├ 参拝・信仰    208問      │
│    └ 建造物・美術   66問      │
```

展開例（歴史・人物をタップ）:
```
│  📜 歴史・人物    176問       │
│    ├ すべて        176問      │
│    ├ 参詣文化       ~35問     │  ← 金毘羅参り、五街道、代参
│    ├ 人物          ~55問     │  ← 燕石、琴陵、邦坊 etc.
│    ├ 宮の由緒      ~35問     │  ← 創建、祭神、神仏分離
│    ├ 町の歩み      ~30問     │  ← 町制、鉄道、博覧会
│    └ 伝統行事      ~20問     │  ← 蹴鞠、五人百姓の沿革
```

#### カテゴリ階層の定義

| UIグループ | JSONカテゴリ | サブカテゴリ | 問題数 |
|-----------|------------|------------|--------|
| ⛩️ 金刀比羅宮 | — | — | 274問 |
| | shrine | 参拝・信仰 | 208問 |
| | architecture | 建造物・美術 | 66問 |
| 📜 歴史・人物 | history | — | 176問 |
| | history (※タグで分割) | 参詣文化 | ~35問 |
| | history (※タグで分割) | 人物 | ~55問 |
| | history (※タグで分割) | 宮の由緒 | ~35問 |
| | history (※タグで分割) | 町の歩み | ~30問 |
| | history (※タグで分割) | 伝統行事 | ~20問 |
| 🎭 金丸座・歌舞伎 | theater | — | 55問 |
| 🗺️ 地理・観光 | — | — | 138問 |
| | geography | 地理・行政 | 86問 |
| | tourism | 観光スポット | 52問 |
| 🍜 グルメ・名産品 | — | — | 64問 |
| | gourmet | 讃岐うどん・名店 | 44問 |
| | food | お菓子・地酒 | 20問 |
| 🎨 文化・伝統 | culture | — | 68問 |
| 🎪 祭り・イベント | event | — | 82問 |
| 🏘️ 現代の琴平 | — | — | 91問 |
| | modern | まちの取り組み | 47問 |
| | life | 暮らし・施設 | 44問 |

**実装方針:**
- shrine / architecture / geography / tourism / gourmet / food / modern / life はJSON category値でそのまま分割可能
- **history のサブカテゴリは、JSONに `subcategory` フィールドを追加してタグ付けする**（後日データ改修で対応）
  - タグ値: `pilgrimage`（参詣文化）, `person`（人物）, `shrine_origin`（宮の由緒）, `town`（町の歩み）, `tradition`（伝統行事）
- history のサブカテゴリが未タグの間は、「歴史・人物」はサブ展開なし（全176問から出題）
- テーマえらびで英語は表示しない（英語は「英語もまぜる」モード専用）

### 2.4 クイズ画面（既存を改修）

基本構造は現行を維持。変更点:

- **ヘッダーにモード表示**: 「フリーモード」「easyモード」「金刀比羅宮」など、今どのモードで解いているか表示
- **難易度バッジ**: 各問題に easy/medium/hard のバッジを表示（現行は非表示）
- **カテゴリバッジ**: 現行のcategoryタグ表示は維持

### 2.5 スコア画面（既存を改修）

変更点:

- **「次の10問へ」ボタンを追加**: 同じモード設定で次の10問を即開始
- **「モードを変えて遊ぶ」ボタン**: スタート画面に戻る
- **難易度別の正答率を表示**（easy/medium/hardそれぞれの成績）
- カテゴリ別のスコアは現行の琴平/英語の2分割から、カテゴリ別に細分化

## 3. 出題ロジック

### 3.1 フリーモード

- 琴平問題のみ10問
- 全カテゴリ・全難易度からランダム
- カテゴリ分散: 各カテゴリ最大2問（現行の最大1問から緩和し、多様性を保ちつつ偏りすぎを防ぐ）
- 同じ日に同じ問題は出さない（localStorage）

### 3.2 難易度えらびモード

- 選択した難易度のみから10問出題
- カテゴリ分散は適用
- 「おまかせ」: 出題時にランダムで構成を決定
  - 例: easy2 + medium5 + hard3、easy4 + medium4 + hard2 など

### 3.3 テーマえらびモード

- 選択したUIグループまたはサブカテゴリのみから10問出題
  - 例:「金刀比羅宮 > すべて」→ shrine + architecture から10問
  - 例:「金刀比羅宮 > 建造物・美術」→ architecture のみから10問
  - 例:「歴史・人物」→ history 全体から10問（サブカテゴリ未タグの間）
- 難易度はランダム（プール内の比率に従う）
- カテゴリ分散は適用しない（テーマ絞り込みが目的なので）
- プールが10問未満の場合はプール全問を出題

### 3.4 英語もまぜるモード

- 現行と同じ: 琴平5問 + 英語5問

### 3.5 共通ルール

- 出題順はシャッフル（毎回ランダム）
- `enabled: true` かつ `reviewed: true` の問題のみ
- 同一セッション内での重複なし
- 連続プレイ時は前ラウンドの問題を除外

## 4. 段階的リリース計画

### Phase A: 最小限の改修（最優先）

1. **スタート画面の追加** — モード選択UI
2. **「琴平のみ10問」をデフォルトに** — 英語はオプション化
3. **「次の10問へ」ボタン** — スコア画面に追加
4. **難易度バッジの表示** — 各問題にeasy/medium/hard表示

### Phase B: モード拡充

5. **難易度えらびモード** — easy/medium/hard/おまかせ選択
6. **テーマえらびモード** — カテゴリ選択

### Phase C: 体験向上

7. **スコア画面の改善** — 難易度別・カテゴリ別の正答率表示
8. **成績履歴画面** — 過去のスコア一覧（localStorageから）
9. **間違えた問題の復習モード**

## 5. 技術的な方針

- **ルーティングは不要**（Phase 1の範囲ではSPA内のstate遷移で十分）
- **状態管理**: useQuizフックにモード設定を追加パラメータとして渡す
- **URLクエリパラメータ**: `?mode=easy` 等でモード直指定できると共有に便利（Phase B以降）
- **レスポンシブ**: モバイルファースト維持。スタート画面のボタンは指で押しやすいサイズ
- **データ読み込み**: 現行のfetch + cacheの仕組みを維持。フィルタリングはフロント側で実施
